# Konum Takip Backend

## Proje Açıklaması
Bu proje kullanıcıların konum verilerini işleyen bir backend uygulamasıdır.

## Kullanılan Teknolojiler
- Node.js
- NestJS
- Prisma
- PostgreSQL

## Özellikler
- Konum verisi alma
- Veritabanına kaydetme
- API ile veri sunma

## Kurulum

npm install  
npm run start

## Not
Bu proje backend servisidir ve frontend ile birlikte çalışır.
